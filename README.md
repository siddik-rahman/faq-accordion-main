# Frontend Mentor - FAQ accordion

![Design preview for the FAQ accordion coding challenge](preview.jpg)

## Description  
This is a simple FAQ accordion challenge from Frontend Mentor. 
this project demonstrates the use of semantic HTML, TailwindCSS, and JavaScript 
to create an accessible and interactive user experience.
---  
## Overview 
- View an optimized layout across different screen sizes.
- Expand and collapse FAQ items.
- See hover and focus states on all interactive elements.
- Navigate the accordion using keyboard controls.
## Features 
- Responsive design for different screen sizes.
- Accordion component that alows users to expand and collapse sections.
- Clean and user-friendly interface.  
---  
## Built with
- HTML5
- TailwindCSS
- JavaScript(ES6)
- Mobile-first workflow
